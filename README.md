# Freelancing Company Website

A modern React + Tailwind CSS website inspired by www.myeglu.com, featuring a blue and purple gradient theme, multiple pages, and backend integration with MySQL.

## Features
- Home, Services, About, Contact pages
- Responsive design with gradient theme
- Backend API using Express and MySQL

## Setup

### Frontend
1. Install dependencies:
	```bash
	npm install
	```
2. Start the development server:
	```bash
	npm run dev
	```

### Backend
1. Install backend dependencies:
	```bash
	npm install express mysql cors
	```
2. Set up MySQL database and update credentials in `server.js`.
3. Start the backend server:
	```bash
	node server.js
	```

## Customization
- Update gradient colors in Tailwind config or page components as needed.
- Add more pages or features as required.
